HTML5 Opmaak

"Oefening baart kunst" zeggen ze altijd. Wel dat kan ik bevestigen!

Oefenen, oefenen en nog eens oefenen!
Het internet, en dan zeker de internettaal HTML5 past zich steeds aan, waardoor je als webdeveloper steeds opnieuw de laatste trends moet volgen.
Bijscholen, en veel opzoekwerk, is dan ook een noodzaak!

CSS3 Styling

Ook bij CSS (Cascading Style Sheets) is het een noodzaak om steeds up-to-date te zijn. CSS is niet enkel spelen met kleur of lettertype. Het is héél veel meer dan dat!
De layout van een website is super belangrijk geworden. Want we willen nu eenmaal dat onze website netjes wordt getoond op de verschillende types schermen.
Dus "Responsive Design" is een absolute must!

Grafisch Design

En dan hebben we nog het visuele aspect. De HTML en de CSS mogen dan super goed zijn, maar wat ben je ermee als de afbeeldingen die gebruikt worden op jouw website niet om aan te zien zijn?
Ook hiermee kan ik je op weg helpen. Ik ben géén wizard op grafisch gebied, maar ik trek me zeker uit de slag, zodat de afbeeldingen op elk type toestel correct worden weergegeven. En de gebruiker van jouw website een aangename ervaring krijgt.
